# Interactive Prototype Creation

This project contains an interactive mobile app prototype designed using Figma.

## Features
- Clickable mobile app prototype
- Interactive navigation flow
- Modern UI/UX screens
- Smooth screen transitions
- User-friendly interface

## Tool Used
- Figma

## Prototype Details
The prototype was created to simulate real mobile app interactions including navigation between screens, user flow, and interactive components.

## User Testing
Basic user testing was conducted to check usability, navigation clarity, and interaction flow. Improvements were made based on feedback.
